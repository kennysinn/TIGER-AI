# Tiger AI Website

Static first-version website for Tiger AI, an AI Company Builder Platform for solopreneurs and small agencies.

## Current Version

This version is designed for GitHub Pages and uses plain HTML, CSS, and JavaScript.

Included:

1. Product positioning page
2. Login modal prototype
3. AI company template cards
4. AI employee dashboard preview
5. Business kit and demo link previews
6. Product planning document

## Open Locally

Open `index.html` directly in a browser.

## GitHub Pages Deployment

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings > Pages.
4. Set Source to `Deploy from a branch`.
5. Select `main` and `/root`.
6. Save.

The site will be available at:

`https://YOUR_USERNAME.github.io/YOUR_REPOSITORY_NAME/`
